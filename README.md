# Finance-Report-Dashboard
Power BI Finance Dashboard analyzing sales, profit, products and country-wise performance.

# Project Overview
This project presents an interactive Finance Dashboard built in Power BI to analyze sales, profit, products and country-wise performance.

# Objectives
- Analyze sales and profit performance
- Compare product performance
- Track profit by country
- Understand sales by customer segments
- Monitor monthly profit trends

# Tools Used
- Power BI
- Excel
- DAX

# Key KPIs
- Gross Sales: 127.93M
- Total Sales: 118.73M
- Profit: 16.89M
- Units Sold: 1.13M
